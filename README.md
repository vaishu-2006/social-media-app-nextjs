# Social Media Web Application

A modern social media platform built using Next.js, TypeScript, and React.

## Features

* User Authentication
* Create and Manage Posts
* User Profiles
* Reels Section
* Notifications
* Saved Posts
* Responsive UI Design

## Tech Stack

* Next.js
* TypeScript
* React
* Tailwind CSS
